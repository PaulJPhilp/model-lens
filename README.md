
# ModelLens

ModelLens is an open-source web dashboard for AI engineers to explore and visualize LLM models.

## Features

- Interactive table for filtering/sorting models
- On-the-fly token/cost estimations with @tokenlens/tokenlens
- Timeline view for model evolution

## Tech Stack

- Next.js 15 (App Router)
- Effect 3.1.9
- TanStack React Table
- Vis.js
- shadcn/ui
- Tailwind CSS 4
- TurboRepo
- pnpm

## Setup

1. `pnpm i`
2. `pnpm run dev`
3. Open [http://localhost:3000/models](http://localhost:3000/models)

## Deployment

Deploy to Vercel by connecting your GitHub repository.
